# Modaics iOS App

SwiftUI iOS application for Modaics.

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+

## Structure

```
iOS/Modaics/
├── App/              # App entry and configuration
├── DesignSystem/     # Colors, typography, components
├── Core/             # Utilities, extensions
├── Presentation/     # Views and ViewModels
└── Services/         # API clients, repositories
```

## Setup

1. Open `Modaics.xcodeproj` in Xcode
2. Select your development team
3. Build and run (Cmd+R)

See main README.md for detailed instructions.
